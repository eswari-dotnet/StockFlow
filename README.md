Developed a RESTful Web API using ASP.NET Core (.NET 6/7) for real-world backend applications.
Implemented CRUD operations with SQL Server using Entity Framework Core.
Followed clean architecture using Repository Pattern and Dependency Injection.
Used DTOs with AutoMapper for secure and efficient data transfer.
Applied Fluent Validation and Data Annotations for input validation.
Handled errors globally using proper exception handling techniques.
Implemented JWT-based authentication and role-based authorization for security.
Added filtering, pagination, logging, and response caching for performance improvement.
Supported API versioning for better maintainability and scalability.
Designed the project to be production-ready with deployment support on Azure and IIS.
